# AlzherimersDetection
A system that detects the presence of Alzheimer's in MRI images

Data set : http://adni.loni.usc.edu/data-samples/access-data/

Dependencies :

Keras >= 2.2.5

matplotlib >= 3.0.0

mxnet >= 1.6.0b20190724

nilearn >= 0.5.1

Numpy >= 1.16.5

Pandas >= 0.25.0

Python >= 3.5

scikit-learn >= 0.21.3

scikit-misc >= 0.1.2

scipy >= 1.4.0

tensorflow >= 1.13.1
